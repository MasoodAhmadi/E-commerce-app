# E-commerce-app

This is an e-commerce application that a user can order a product and buy stuffs.

## Installation

add .env file

- SECRETKEY = xxxxxxxxx;
- DB_PASSWORD=xxxxxxxxx
- DB_USERNAME=xxxxxxx
- DB_DATABASE=xxxxxx

## client

```bash
npm install && npm start
```

## server

```bash
npm install && npm run dev
```
